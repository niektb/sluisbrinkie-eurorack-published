This is a version of MI braids for Toepler+ with CV input dealing with note input. It's a little flaky, but can be tuned.

Requires the following Arduino libraries:

 * STMLIB.h https://github.com/poetaster/STMLIB
 * BRAIDS.h https://github.com/poetaster/BRAIDS
 * Bounce2.h
 * RPi_Pico_TimerInterrupt.h
