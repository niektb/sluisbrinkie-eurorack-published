This is a version of MI braids, plaits, rings and clouds in a single module 

Requires the following Arduino libraries:

 * STMLIB.h https://github.com/poetaster/STMLIB (plaits1.2 branch)
 * BRAIDS.h https://github.com/poetaster/BRAIDS
 * BRAIDS.h https://github.com/poetaster/PLAITS (plaits1.2 branch)
 * BRAIDS.h https://github.com/poetaster/RINGS
 * BRAIDS.h https://github.com/poetaster/Clouds
 * pio_encoder.h
 * Bounce2.h
 * RPi_Pico_TimerInterrupt.h (working on removing this)
 * Adafruit_SSD1306.h
